# Pikmin Turntable 轉盤計時工具

《Pikmin Bloom》轉盤（六個花盆）點擊時機練習工具。開啟頁面、輸入起始秒數、按下開始，六個花盆會依序在對應的時間點亮，協助練習點擊節奏。

## 使用方式

直接開啟網頁即可使用，免安裝：

**https://evilpiyo.github.io/Pikmin-Turntable/**

或掃描 QR Code：

![QR Code](_Ref/pikmin-turntable-qr.png)

## 玩法說明

點擊卡片可切換盆栽類型（🔮其他／👑金盆／🥙大花苗），僅影響顯示圖示，不影響計時。計時規則參考自 Pikmin Bloom 台灣社群討論的觀察心得（`_Ref/規則心得.png`）。
